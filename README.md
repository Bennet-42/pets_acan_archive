# 菜瓜cp帖 · 豆瓣小组帖子归档

这是一个豆瓣小组帖子的本地静态归档，使用 GitHub Pages 在线浏览。

- **原帖**：[豆瓣小组 #490119344](https://www.douban.com/group/topic/490119344/)
- **当前已归档**：1440 页（持续更新中）
- **在线阅读**：https://bennet-42.github.io/pets_acan_archive/

## 功能

- 完整保留帖子主体（标题、楼主发言、所有楼层评论、配图）
- 移除豆瓣自带的导航 / 广告 / 隐私弹窗
- 本地分页器：支持「跳到第 N 页」输入框 + 键盘 ←/→ 翻页
- 配图全部本地化，断网也能浏览
- 移动端适配

## 文件结构

```
.
├── index.html        # 首页：跳转入口 + 全部页面索引
├── 1.html            # 第 1 页
├── 2.html            # 第 2 页
├── ...
├── N.html            # 第 N 页
├── images/           # 帖子配图（头像、楼层图片）
└── meta.json         # 元数据：原帖 URL、归档时间
```
